# supply-chain-defense-skills

Reusable Security Skills for AI Coding Agents (Claude Code, Cursor, etc.) to protect against supply chain attacks like Mini Shai-Hulud and similar threats in npm, PyPI and other ecosystems.

## Дневные обзоры аналитика

Структурированные обзоры по вайбкодингу, agentic engineering, безопасности supply chain и Rust + WASM.

- [2026-05-17: Вайбкодинг, Agentic Engineering и Безопасность Supply Chain](docs/daily-reports/2026-05-17-agentic-engineering-security-overview.md)

## Структура репо

- `docs/` — документация и обзоры
- `skills/` — реюзабельные скиллы и playbooks для AI-агентов

## Как использовать

1. Копируй скиллы в свои проекты / промпты для Claude Code / Cursor.
2. Добавляй новые playbooks в `skills/`.
3. Следи за дневными обзорами в `docs/daily-reports/`.

** Цель:** Минимизировать риски supply chain при работе с AI coding agents.
